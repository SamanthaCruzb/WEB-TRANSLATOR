# Cloud Translator Web App

## Description
This project is a web-based translator that uses Microsoft Azure Translator API to translate text in real time.

## Features
- Real-time text translation
- Integration with Microsoft Azure Cognitive Services
- Simple and responsive UI

## Technologies
- JavaScript
- HTML / CSS
- Microsoft Azure API
